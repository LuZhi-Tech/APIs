# 接口介绍：
查询国内城市天气基础数据（天气情况，最低温度，最高温度）

# 示例：
```
查询“贵阳”天气
``` 
```json
{
  "code": 200,
  "message": "Succeed",
  "data": {
    "province": "贵州",
    "cityname": "贵阳",
    "temperature_high": 9,
    "temperature_low": 16,
    "description": "小雨转晴" 
  }
}
```
